# Contributing to asimov

Thank you for your interest in contributing to **asimov**!  
This project provides a structured ethical framework based on Isaac Asimov’s Three Laws of Robotics and the Zeroth Law. We welcome improvements that make the skill clearer, more robust, and more useful.

## Ways to Contribute

You can contribute in several ways:

- Reporting bugs or unclear sections in the skill
- Suggesting improvements to the instructions or edge-case handling
- Improving documentation and examples
- Adding new activation triggers or usage scenarios
- Fixing typos or formatting issues
- Discussing philosophical or practical applications of the laws

## Reporting Issues

If you find a problem or have a suggestion:

1. Check if the issue has already been reported.
2. Open a new issue with a clear title and description.
3. Include examples of where the current behavior is unclear or problematic.
4. For philosophical or edge-case discussions, feel free to open a discussion first.

## Pull Requests

We welcome pull requests! Here’s how to make the process smooth:

1. **Fork** the repository and create a new branch for your changes.
2. Make your changes focused and clear.
3. Update the `SKILL.md` and/or documentation as needed.
4. Ensure your changes align with the spirit and structure of the original framework.
5. Submit a pull request with a good description of what you changed and why.

### Guidelines for Skill Improvements

- Stay faithful to Asimov’s original laws and their hierarchical priority.
- Focus on **clarity**, **safety**, and **practical usability**.
- When proposing changes to core reasoning steps, explain the reasoning clearly.
- Avoid adding new laws or fundamentally altering the priority order unless there is strong justification.
- Keep language precise and professional.

## Naming Policy

This project is called **asimov**.  

If you create a significantly modified or forked version, please use a **different name** for your project. This helps avoid confusion in the community. You are welcome to mention that your work is inspired by or derived from this project.

## Code of Conduct

Be respectful and constructive in all interactions.  
We follow the spirit of the [Contributor Covenant](https://www.contributor-covenant.org/). Harassment or disrespectful behavior will not be tolerated.

## Questions?

If you have questions about contributing or the project in general, feel free to open an issue or start a discussion. We’re happy to help!
